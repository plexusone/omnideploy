## [v0.1.0] - 2026-06-22

### Highlights

- Universal deployment tool for container applications with declarative YAML configuration
- AWS LightSail container service support with Pulumi IaC backend
- ECR repository management with build and push workflow

### Added

- Core interfaces and configuration schema for deployment targets, backends, and runtimes ([`311447d`](https://github.com/plexusone/omnideploy/commit/311447d))
- AWS LightSail container service target with automatic container definition generation ([`0ffa07e`](https://github.com/plexusone/omnideploy/commit/0ffa07e))
- Pulumi IaC backend for infrastructure state management ([`de73285`](https://github.com/plexusone/omnideploy/commit/de73285))
- OmniAgent and generic container runtime adapters ([`3550a5c`](https://github.com/plexusone/omnideploy/commit/3550a5c))
